# Technology Stack

## Core Technologies
- **Programming Language:** Python
- **GUI Framework:** PyQt6

## Audio & Processing
- **Audio Capture:** PyAudio
- **Audio Processing:** NumPy, SoundFile
- **AI Integration:** Groq (Python Client) for transcription and Llama/Mixtral inference.

## Utilities
- **Global Hotkeys:** Keyboard library
- **Clipboard Management:** Pyperclip

## Development & Testing
- **Testing Framework:** Pytest
- **Coverage:** Pytest-cov
- **UI Testing:** Pytest-qt
- **Mocking:** Pytest-mock, Unittest.mock
